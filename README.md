# Version Pinning

A gem for pinning versions from Gemfile.lock to Gemfile.

### Prerequisites

Ruby 3.2 or compatible
Bundler 2.6.1 or compatible

### Structure

```
.
├── version_pinning.gemspec
├── .gitignore
└── lib
    └── version_pinning.rb
└── spec
    └── factories
        └── base_gemfile_locks.rb
    └── version_pinning_spec.rb
```
